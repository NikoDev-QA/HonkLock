# HonkLock
🦆🔒 HonkLock: Antirrobo Open Source. Detecta modo avión, extracción de SIM e intentos de apagado, bloqueando la pantalla y forzando una alarma inesquivable. | Open Source Anti-theft. Detects airplane mode, SIM removal &amp; shutdown attempts, locking the screen and forcing an unbypassable alarm. (Built with Kotlin &amp; Jetpack Compose).
